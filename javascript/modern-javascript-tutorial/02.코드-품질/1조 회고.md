### 1. thisArg는 this 바인딩할 때쓰는건가요?

네 맞습니다. this가 동적으로 바인딩이 되는데 메서드에서 호출하면 메서드를 호출한 객체를 this가 가리키고, 생성자에서 this를 사용하면 생성한 인스턴스를 가리키고 그러는데, call 메서드에 this 바인딩을 하지 않으면 여기서의 this는 전역객체를 가리키게 되기 때문에 바인딩을 위해 .call을 사용함
